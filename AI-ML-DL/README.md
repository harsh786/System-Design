## How AI, ML, DL, and Data Science Relate

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ARTIFICIAL INTELLIGENCE (AI)                       │
│  "Any technique that enables machines to mimic human behavior"       │
│                                                                      │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              MACHINE LEARNING (ML)                            │   │
│   │  "Subset of AI that learns from data without being           │   │
│   │   explicitly programmed"                                     │   │
│   │                                                              │   │
│   │   ┌─────────────────────────────────────────────────────┐   │   │
│   │   │            DEEP LEARNING (DL)                        │   │   │
│   │   │  "Subset of ML using neural networks with            │   │   │
│   │   │   multiple layers"                                   │   │   │
│   │   └─────────────────────────────────────────────────────┘   │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                      │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              DATA SCIENCE                                    │   │
│   │  "Interdisciplinary field using scientific methods,          │   │
│   │   algorithms, and systems to extract knowledge from data"    │   │
│   │   (Overlaps with ML but also includes data engineering,      │   │
│   │    visualization, statistics, and domain expertise)          │   │
│   └─────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────┘
```

## Prerequisites

- Basic programming knowledge (Python preferred)
- High school mathematics
- Curiosity and willingness to learn

## Setup

```bash
# Create a virtual environment
python3 -m venv ai-ml-env
source ai-ml-env/bin/activate

# Install all required packages
pip install numpy pandas matplotlib seaborn scikit-learn
pip install torch torchvision torchaudio
pip install tensorflow keras
pip install jupyter notebook
pip install mlflow wandb
pip install fastapi uvicorn
pip install plotly bokeh
```
