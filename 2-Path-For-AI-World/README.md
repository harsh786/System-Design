## How To Use This Path

Read the files in numeric order. Each stage has a different job:

|---|---|---|

## Mastery Gates

| Level | You are ready to move on when... |
|---|---|
| Beginner | You can build authenticated, observable backend APIs with tests, Docker, Postgres, Redis, and basic LLM integrations. |
| Core | You can build a measurable RAG system with ingestion, chunking, hybrid retrieval, reranking, citations, ACLs, and retrieval evals. |
| Advanced | You can design bounded agents that plan, call tools, verify evidence, track confidence, abstain, and escalate. |
| Production | You can deploy AI systems with eval gates, observability, guardrails, AI gateway controls, cost tracking, SLOs, and rollback. |
| Enterprise | You can design reusable AI platforms, governance models, registries, risk controls, multi-tenant scale, and executive-ready architecture docs. |
