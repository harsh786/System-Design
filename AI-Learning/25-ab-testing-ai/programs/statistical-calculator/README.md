# Statistical Calculator

Calculate statistical significance, sample sizes, and confidence intervals for AI A/B experiments.

## Features

- Sample size calculation (how many observations do you need?)
- P-value computation for experiment results
- Confidence interval calculation
- Power analysis
- 5 example scenarios with different effect sizes

## Usage

```bash
pip install -r requirements.txt
python main.py
```

## What It Demonstrates

- When you have enough data to make a decision
- How effect size impacts required sample size
- "You need N more samples to reach significance"
- Different statistical tests for different metric types
