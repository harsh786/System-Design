# Results Analyzer

Analyze A/B experiment results and make ship/no-ship recommendations.

## Features

- T-test and chi-squared significance testing
- Confidence interval calculation
- Multi-metric analysis (primary + guardrails)
- Automated ship/no-ship recommendation with reasoning
- Text-based experiment report with charts

## Usage

```bash
pip install -r requirements.txt
python main.py
```

## What It Demonstrates

- How to analyze experiment results systematically
- Multi-metric decision framework in action
- When to ship, investigate, or discard
- Clear reporting format for stakeholders
