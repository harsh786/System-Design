# Token Counter

A hands-on program that demonstrates how tokenization works across different models.

## What You'll Learn

- How text is broken into tokens
- Why the same text produces different token counts for different models
- How to calculate the cost of any text for any model
- Visual representation of token boundaries

## Run It

```bash
pip install -r requirements.txt
python main.py
```

## Try These Experiments

1. Compare English vs non-English text token counts
2. See how code vs prose differs in token efficiency
3. Calculate the cost of sending your entire README as context
