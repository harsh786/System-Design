# Quantization Comparison

Simulates quantization effects on model memory, speed, quality, and cost.

## What It Does

- Calculates memory for FP16/INT8/INT4 across model sizes
- Simulates quality degradation at different precision levels
- Computes cost savings from quantization
- Provides recommendations based on requirements

## Run

```bash
pip install -r requirements.txt
python main.py
```
