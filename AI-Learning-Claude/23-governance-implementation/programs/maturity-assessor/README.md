# AI Platform Maturity Assessor

Interactive tool that assesses your organization's AI platform maturity level (L0-L5) across governance, evaluation, operations, and security dimensions.

## Features
- 20 assessment questions scored 0-5
- Calculates overall maturity level
- Shows per-category breakdown
- Generates improvement roadmap
- Recommends next steps for reaching the next level

## Usage

```bash
pip install -r requirements.txt
python main.py
```

## Modes
- **Interactive**: answer each question with a score 0-5
- **Quick**: use default scores (demonstrates the tool)
- **Custom**: provide scores as command-line argument
